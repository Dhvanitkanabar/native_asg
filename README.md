# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/smart-field-survey.git
```

Go inside project

```bash
cd smart-field-survey
```

Install dependencies

```bash
npm install
```

Start Expo

```bash
npx expo start
```

---

# 📦 Build APK

```bash
eas build --platform android --profile preview
```

---

# 📱 Build Production

```bash
eas build --platform android --profile production
```
